## üé¨ Demo Tips

### Before Demo:
1. **Pre-load demo data** - Have Sarah's profile ready
2. **Test internet** - Have backup hotspot
3. **Clear browser cache** - Fresh start
4. **Practice timing** - Rehearse 3+ times
5. **Prepare for failures** - Have screenshots ready

### During Demo:
1. **Speak clearly** - Judges need to hear every word
2. **Point to screen** - Guide their eyes
3. **Pause for effect** - Let wow moments sink in
4. **Smile and be confident** - You believe in this
5. **Watch the clock** - Stay within 7 minutes

### After Demo:
1. **Invite questions** - Be ready for technical queries
2. **Have backup slides** - Architecture, tech stack
3. **Know your numbers** - Revenue, users, costs
4. **Be honest** - If something is mock, say so
